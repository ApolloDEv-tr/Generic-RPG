# RPG Character Creator (Python)

A terminal-based RPG character creator script built in Python. The program tracks user choices for character background, sets algorithmic attribute scaling based on class mechanics, and handles dynamic state modification via a multi-tiered difficulty system.

---

## Features

- Choose a character race (Human, Elf, Dwarf, Angel)
- Choose a class (Warrior, Mage, Tank, Archer, Thief)
- Choose difficulty (Easy, Medium, Hard, Extreme)
- Health system that changes based on class and difficulty
- Simple dictionary-based character storage

---

## How it works

The program:
1. Asks for your name
2. Lets you pick a race with special abilities
3. Lets you pick a class with different health values
4. Lets you choose difficulty which affects health
5. Stores everything in a `player` dictionary
